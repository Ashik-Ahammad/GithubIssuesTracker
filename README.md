## 🚀 API Endpoints

| Action | Endpoint | Example |
| :--- | :--- | :--- |
| **Get All Issues** | `https://phi-lab-server.vercel.app/api/v1/lab/issues` | [View All](https://phi-lab-server.vercel.app/api/v1/lab/issues) |
| **Single Issue** | `https://phi-lab-server.vercel.app/api/v1/lab/issue/{id}` | [Issue #33](https://phi-lab-server.vercel.app/api/v1/lab/issue/33) |
| **Search Issue** | `https://phi-lab-server.vercel.app/api/v1/lab/issues/search?q={searchText}` | [Search Example](https://phi-lab-server.vercel.app/api/v1/lab/issues/search?q=notifications) |

### Usage

**Fetch Single Issue:**
`GET https://phi-lab-server.vercel.app/api/v1/lab/issue/33`

**Search Issues:**
`GET https://phi-lab-server.vercel.app/api/v1/lab/issues/search?q=notifications`
