# JavaScript ES6 Queries 

---

### 🟢 1. var, let, and const
`var` is the old way that can sometimes create confusion in the code, so modern developers avoid it. Use `let` when your data needs to change, and `const` for data that stays the same forever. `var` is old and allows reassignment. `let` is scoped and prevents unintended side effects. `const` is immutable but allows array changes.

---

### 🟠 2. The Spread Operator (`...`)
The spread operator is a shortcut that **"unpacks"** all elements from an array or object into a new place. It is the easiest way to copy or combine the data without manually typing every single item.

---

### 🔵 3. map(), filter(), and forEach()
`map()` creates a new list by changing every item, while `filter()` creates a new list by only keeping the items you want. `forEach()` is different because it just runs a task for every item but doesn't give us a new list back.

---

### 🟡 4. Arrow Functions
Arrow functions are a shorter and more modern way to write functions using the `=>` symbol instead of the `function` keyword. It make our code look much cleaner and are very easy to read for beginners.

---

### 🟣 5. Template Literals
Template literals use **backticks** (`` ` ``) instead of quotes to let us easily mix text and variables together. We just put our variable inside `${}` and it fits perfectly into the string without using any plus signs.
